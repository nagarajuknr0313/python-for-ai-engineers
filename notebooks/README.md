# Notebook playground

You can open these notebooks in GitHub Codespaces or Jupyter for a browser-based Python experience.

- Use the provided devcontainer to launch a ready-to-run environment.
- Open the notebook and execute each concept interactively.
- Start with the main playground notebook and then try the lesson files in the content folder.

## Quick start
1. Open the repository on GitHub.
2. Click the Codespaces badge in the README.
3. Wait for the environment to build.
4. Open [concept_playground.ipynb](concept_playground.ipynb) and run the cells.
