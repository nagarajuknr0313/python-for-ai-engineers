# Level 1: Python Fundamentals

This level introduces Python through AI engineering scenarios.

## Topics and Code Files
1. Variables using AI examples: [ai_variables.py](ai_variables.py)
2. Data types using embeddings and vectors: [embeddings_and_vectors.py](embeddings_and_vectors.py)
3. Lists using chat history: [chat_history_lists.py](chat_history_lists.py)
4. Dictionaries using prompt templates: [prompt_templates_dicts.py](prompt_templates_dicts.py)
5. Sets using unique tokens: [unique_tokens_sets.py](unique_tokens_sets.py)
6. Tuples using model configurations: [model_config_tuples.py](model_config_tuples.py)
7. Conditions using model selection: [model_selection_conditions.py](model_selection_conditions.py)
8. Loops using batch inference: [batch_inference_loops.py](batch_inference_loops.py)
9. Functions using prompt pipelines: [prompt_pipelines.py](prompt_pipelines.py)
10. Classes using AI agent examples: [ai_agent_classes.py](ai_agent_classes.py)

## Unique Approach
Each concept is taught through a new AI product scenario: a system that tracks model experiment metadata, prompt variants, and agent state.

## Suggested Learning Order
1. Study the concept explanation in the matching markdown file.
2. Run the example code in the corresponding Python file.
3. Complete the exercises.
4. Attempt the challenge problems.
