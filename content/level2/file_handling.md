# File Handling for Training Data

## Concept explanation
File handling lets a program read from and write to local files. In AI work, this is essential when working with training examples, evaluation logs, or prompt collections.

## Real-world AI use case
A machine learning team may store annotated examples in a CSV file and load them into a training pipeline for analysis.

## Execution flow
1. Open a data file from disk.
2. Read the rows into memory.
3. Process each row.
4. Save the cleaned output to a new file.

## Exercises
- Add a new sample row to the CSV file.
- Save the processed rows as JSON.
- Print the number of loaded rows.
