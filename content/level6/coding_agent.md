# Coding Agent

## Concept explanation
A coding agent performs software tasks such as implementing features, editing files, or explaining code changes.

## Real-world AI use case
A developer assistant can help generate a small module, explain a failing test, or suggest refactors.

## Execution flow
1. Receive a task request.
2. Inspect the code context.
3. Suggest or apply a change.
4. Verify the result with testing.

## Exercises
- Create a simple coding task for the agent.
- Ask the agent to explain a function.
- Add a validation step after the change.

## Interview questions
- How would a coding agent differ from a general assistant?
- Why is validation important in coding workflows?
- What tool access would be useful for a coding agent?
