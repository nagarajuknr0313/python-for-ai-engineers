# Embeddings and Similarity Search

## Concept explanation
Embeddings represent ideas as vectors so systems can compare them numerically. Similarity search finds items whose vectors are closest to a query vector.

## Real-world AI use case
A knowledge assistant can search a document library by comparing the user query embedding to stored document embeddings.

## Execution flow
1. Create a query vector.
2. Create one or more candidate vectors.
3. Compute similarity scores.
4. Rank the results by relevance.

## Exercises
