# Prompt Templates

## Concept explanation
Prompt templates structure instructions so an AI system can fill in dynamic values while preserving consistency.

## Real-world AI use case
A customer support bot can use a reusable template to ask for account details and summarize the issue.

## Execution flow
1. Define a template string.
2. Fill in the placeholders.
3. Send the final prompt to the model.
4. Review the response.

## Exercises
