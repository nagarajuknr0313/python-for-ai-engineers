# Multi-Agent Communication

## Concept explanation
Multi-agent communication allows several agents to share information and collaborate on a task.

## Real-world AI use case
A research team can use one planner agent and one writing agent to divide work between planning and drafting.

## Execution flow
1. Create two or more agents with different roles.
2. Let one agent produce findings.
3. Pass the findings to another agent.
4. Combine the results into a final response.

## Exercises
- Add a third agent for critique.
- Create a shared message queue conceptually.
- Explain how collaboration improves task quality.

## Interview questions
- Why is coordination important for multi-agent systems?
- What problems can arise from poor communication?
- How would you structure messages between agents?
